# Session 2026-04-13-0001

## 프로젝트 개요
`apt-legal-mcp` — 한국 공동주택 법률 자문 Vertical AI Agent(A2A 지원)가 호출하는 MCP 서버.
국가법령정보센터 Open API를 1차 데이터 소스로 사용하여 법령 조문·판례·행정해석 조회 도구를 제공한다.
파이프라인: ChatGPT Enterprise CustomGPT → Legal Vertical Agent(별도 리포) → 본 MCP 서버 → law.go.kr.

## 중요 참고사항
- **개발 환경**: Windows 11 + Citrix VDI + PowerShell (WSL2 불가). Python 3.14 네이티브, Docker는 원격 빌드 전략.
- **네이밍 일관성**: 코드상 `apt_legal_mcp`(Python 패키지), 프로덕트명 `apt-legal-mcp`, URI 스킴 `apt-legal://`.
- **폴더명 변경**: 사용자가 추후 `aptlaw-mcp` → `apt-legal-mcp`로 폴더를 직접 rename 예정 (세션 종료 후).
- **Citrix VDI 제약**: Hyper-V 중첩 가상화가 막혀있을 가능성 → Docker Desktop 대신 CI 또는 원격 빌드 서버에서 이미지 빌드.
- **판례 XML 필드명**: `판례일련번호`, `사건번호`, `사건명`, `법원명`, `선고일자`, `사건종류명`, `판시사항`, `판결요지`, `판례내용`, `주문`, `참조조문` — 모두 추정이므로 실제 응답과 다를 수 있음.
- **law.go.kr 인코딩**: 응답이 UTF-8/EUC-KR 혼용 가능 → `law_api.py._get_xml`에서 폴백 디코딩 구현됨.
- **캐시 TTL**: 법령 검색 24시간, 조문 전문 7일, 판례 상세 7일, 스테일 캐시 7일(법령 API 장애 대응).
- **`search_interpretation` 스텁**: 현재는 빈 결과 + "연동 대기" 메시지 반환. API 키 확보 즉시 교체 필요.
- **GitHub 리포**: https://github.com/youngs75/apt-legal-mcp (public, main).
